# Python Email Validator

A Python project that validates email addresses using manual string validation logic (without regex).

## Features

- Checks minimum length
- Ensures only one '@' symbol
- Prevents spaces
- Validates domain structure
- Allows only valid characters
- Verifies top-level domain length

## How to Run

Run the file using:

python basic_email_validator.py

## Future Improvement

Regex-based validation will be added in the next update.
